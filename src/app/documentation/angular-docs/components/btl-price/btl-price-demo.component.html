BtlPriceComponent COMPONENT

Nom BtlPriceComponent
Selecteur <btl-price>
Source src/baseBtl/base/btl-price/btl-price.component.ts
Module BaseModule

Composant prix du site.

Utilisation

<btl-price [price]="32.99" [monthly]="true" [promoPrice]="20.99" [position]="'right'"></btl-price>

Attributs

[price]OBLIGATOIRE
number		 Il s'agit du prix principal. Seul attribut obligatoire. L'absence de cet attribut log une erreur en console. Toujours s'assurer de le valoriser
[promoPrice]
number		 Prix promotionnel. Si cet attribut est valorisé, il devient le prix principal et l'attribut price devient le prix secondaire.
[showPromoPrice]
boolean		 Définit si le prix secondaire doit s'afficher ou pas. Par défaut à true. Si on le set à false, le prix secondaire (uniquement s'il s'agit du promoPrice) ne s'affichera pas.
[position]
string		 Positionne le prix secondaire autour du prix principal. Valeurs autorisées :
  • left : valeur par défaut. Le prix secondaire sera positionné à gauche du prix principal.
  • top : Le prix secondaire sera positionné au dessus du prix principal.
  • right : Le prix secondaire sera positionné à droite du prix principal.
[oldPrice]
number		 Permet d'afficher un prix secondaire. Contrairement au promoPrice, il ne tient pas compte de l'attribut showPromoPrice et ne remplacera pas le prix principal. Il s'affichera forcement s'il est valorisé. Exemple d'utilisation: Mur de téléphone B&You
[monthly]
boolean		 Définit s'il s'agit d'un prix mensuel ou non. Par défaut à false
