Formulaires

Voici plusieurs éléments qui peuvent être utilisés dans nos formulaires.
Pour utiliser ce composant il faut que le parent du formulaire contienne la classe .form-container.

Source CSS @import '/scss/components/form';

Button group

<div class="form-container">
    <div class="btn-group" role="group" aria-label="Basic example">
        <label class="btn btn-input-group" for="radio1" [ngClass]="{'btn-outline-info': sampleButtonGroup.value !== '0','btn-info': sampleButtonGroup.value === '0'}">
            <input class="form-control" hidden type="radio" id="radio1" value="0" name="sampleButtonGroup" [formControl]="sampleButtonGroup">
            Monsieur
        </label>
        <label class="btn btn-input-group" for="radio2" [ngClass]="{'btn-outline-info': sampleButtonGroup.value !== '1','btn-info': sampleButtonGroup.value === '1'}">
            <input class="form-control" hidden type="radio" id="radio2" value="1" name="sampleButtonGroup" [formControl]="sampleButtonGroup">
            Madame
        </label>
    </div>
</div>
