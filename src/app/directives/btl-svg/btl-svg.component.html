BtlSvg DIRECTIVE

Nom SvgDirectiveBase
Source src/baseBtl/base/directives/svg.directive.ts
Module BaseModule

Directive servant à l'affichage des icones SVG.

Comment l'utiliser

Appliquer la directive btlSvg à un <span>. Elle doit être accompagnée d'attributs obligatoires qui sont :

  • file : string : Nom du fichier SVG contenant l'icone que l'on souhaite afficher.
  • name : string : Id de l'icone souhaitée.
  • class : string : La classe globale icon.

Exemple : <span btlSvg file="elig-tlv-defs" name="shape-icon-arrow-l" class="icon s-arrow-l"></span>
